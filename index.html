This is an applet I wrote in one night called 'Scrolly1' which renders some pretty scenery by making a lot of expensive calls to AWT functions (such as to paint semi-transparent gradients to simulate fog).

Keys:

  • H - Toggle anti-aliasing on/off
  • G - Cycle through detail modes
  • F - Change fog opacity
  • B - Change fog brightness
  • L - Switch between normal / simplified / no skyscraper window lights
  • arrow keys - accellerate the camera
  • space bar - immediately halt camera movement

You will probably need to click on the applet before keys work. There are some features not documented, here. The jar can also be run stand-alone.
